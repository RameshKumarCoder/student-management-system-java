# Student Management System (Java)

A simple console-based Student Management System developed in Java using Object-Oriented Programming (OOP) concepts.

## Features

- Add Student
- View All Students
- Delete Student
- User-Friendly Console Interface
- Professional Table Format Output

## Technologies Used

- Java
- ArrayList
- Scanner Class
- OOP Concepts

## Project Structure

```
Student-Management-System/
│
├── Main.java
├── Student.java
└── README.md
```

## How to Run

1. Compile the Java files:

```bash
javac Main.java Student.java
```

2. Run the program:

```bash
java Main
```

## Sample Output

```

     STUDENT MANAGEMENT SYSTEM
======================================
1. Add Student

3. Delete Student
4. Exit
======================================
Choose Option:
```
